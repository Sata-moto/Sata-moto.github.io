线性基学习笔记

发布于 2020-02-28 | 分类于 数学 、 知识点 | 10分钟 | 2451字数

这篇文章差点半途咕掉了呢...

前言：

\quad\quad 日常划水的时候无意 getgetget 到了一个好玩的东西：线性基。

\quad\quad 感觉似乎有那么一点点用处，于是记下来供以后参考。

正文：

\quad 构造原理：

\quad\quad 在认识线性基前，我们先要认识 基 是什么。

	在线性代数中，基（也称为基底）是描述、刻画向量空间的基本工具。
														-------百度百科

\quad\quad 学过高中数学的人应该都知道，nnn 维线性空间中任一向量都可以被 nnn 个线性无关的向量表示出来。

\quad\quad 比方说，平面内，以任意两不共线向量作为基底，其它的所有向量都可以被其唯一表示。

\quad\quad 与向量类似，任意一个 nnn 位二进制数都可以由 nnn 个最高位依次不同的二进制数异或得到（性质一，所有性质的证明都放在下一节了）。

\quad\quad 比方说：如果我们有一组二进制数： (1000)2,(0100)2,(0010)2,(0001)2(1000)_2 ,(0100)_2,(0010)_2,(0001)_2(1000)2​,(0100)2​,(0010)2​,(0001)2​（从前到后最高位分别为 4,3,2,14 ,3, 2, 14,3,2,1），那么，任意一个四位二进制数都可以被异或得到（显然）。

\quad\quad 当然，上面那四个数形式优美（可以类比向量中两两垂直的向量），不能作为一般例子看待。

\quad\quad 下面举一个一般一点的例子：(1011)2,(0110)2,(0011)2,(0001)2(1011)_2 ,(0110)_2,(0011)_2,(0001)_2(1011)2​,(0110)2​,(0011)2​,(0001)2​（从前到后最高位分别为 4,3,2,14 ,3, 2, 14,3,2,1），可以发现，这四个数依然可以表示出所有的四位二进制数，因此，这四个数也叫做 所有四位二进制数（这一数集） 的一组线性基。

\quad\quad 线性基的定义：对于数集 SSS ，一定存在数集 PPP，满足从 SSS 中任取一个数 xxx，PPP 中都存在一组异或结果等于 xxx 的数。那么，所有 PPP 中，包含元素的量最少的一组被称作 SSS 的线性基。

\quad\quad 至此，性质一可以更新为： nnn 位二进制数（这一集合）的线性基包含 nnn 个元素。

\quad\quad 注意到，线性基定义中有两个要求：

\quad\quad 第一个是 “完全表示”，即原数组中的所有元素都可以被线性基中的元素异或表示。

\quad\quad 第二个是 “长度最小”，即线性基必须是满足上面那个条件的 元素个数最少的数集。

\quad\quad 如何根据这两个要求构造线性基？

\quad\quad 首先我们要得到性质一的拓展性质：对于一个数集 SSS，一定存在一组所含所有数的二进制最高位都不同的线性基（性质二）。

\quad\quad 因此，对于 SSS 中的每一个元素 xxx，假设其二进制最高位为 KKK，若我们当前线性基中没有二进制最高位为 KKK的元素，我们就将 xxx 放入线性基中并跳出；否则我们把 xxx 与线性基中二进制最高位为 KKK的元素异或的结果作为 xxx 继续处理。

\quad\quad 看起来似乎很麻烦，其实不麻烦，这里举一个例子（以下所有数均为二进制）：

\quad\quad 令 S=1010,1100,0111,0010S={1010,1100,0111,0010}S=1010,1100,0111,0010。

\quad\quad 遍历到第一个数（1010），其最高位为第四位，线性基中没有最高位为第四位的元素，于是把 101010101010 放入线性基中。线性基内元素：1010{1010}1010。

\quad\quad 遍历到第二个数（1100），其最高位为第四位，线性基中有最高位为第四位的元素，于是把 1010⊕11001010 \oplus 11001010⊕1100 的结果作为第二位元素，继续处理第二位元素 。线性基内元素：1010{1010}1010。

\quad\quad 遍历到第二个数（0110），其最高位为第三位，线性基中没有最高位为第三位的元素，于是把 011001100110 放入线性基中。线性基内元素：1010,0110{1010,0110}1010,0110。

\quad\quad 遍历到第三个数（0111），其最高位为第三位，线性基中有最高位为第三位的元素，于是把 0110⊕01110110 \oplus 01110110⊕0111 的结果作为第三位元素，继续处理第三位元素 。线性基内元素：1010,0110{1010,0110}1010,0110。

\quad\quad 遍历到第三个数（0001），其最高位为第一位，线性基中没有最高位为第一位的元素，于是把 000100010001 放入线性基中。线性基内元素：1010,0110,0001{1010,0110,0001}1010,0110,0001。

\quad\quad 遍历到第四个数（1010），其最高位为第二位，线性基中没有最高位为第二位的元素，于是把 001000100010 放入线性基中。线性基内元素：1010,0110,0001,0010{1010,0110,0001,0010}1010,0110,0001,0010。

\quad\quad 构造完毕，设 SSS 中有 nnn 个数，数字最大为 VVV，则构造线性基的复杂度为 O(nlogV)O(nlogV)O(nlogV)（为什么可以这么构造？显然呐，懒得证了）。

\quad\quad 代码：

//S 表示数集，p[i]表示线性基中最高位为i的元素是多少
	for(int k = 1; k <= S.size(); k++)
	{
		for(int i = 63; i >= 0; i--) // 找最高位
			if(S[k] & (1ll << i))
			{
				if(p[i]) //如果线性基中有最高位为i的元素
					S[k] ^= p[i]; //因为S[k]处理后，第i位之前一定都是零，所以没必要跳出重新开始了
				else
				{
					//存起来并跳出
					p[i] = S[k];
					break;
				}
			}
	}

\quad 性质证明：

\quad\quad 性质一：任意一个 nnn 位二进制数都可以由 nnn 个最高位依次不同的二进制数异或得到。

  • \quad\quad 证明：设后面的 nnn 个最高位依次不同的二进制数为 PPP 数集，其中，最高位为 iii 的二进制数表示为 p[i]p[i]p[i]。我们任取一个 nnn 位二进制数 xxx，考虑它的最高位 kkk。我们令 x=p[k]⊕dx=p[k] \oplus dx=p[k]⊕d，显然，ddd 的最高位一定小于 kkk，而如果 ddd 可以被 PPP 中的某些数异或表示，那么 xxx 也可以被表示。因此，我们证明的问题从 一个最高位为 kkk 的 nnn 位二进制数可以被 PPP 表示，变成了 一个最高位为 iii 的 nnn 位二进制数可以被 PPP 表示。显然，后者的问题规模更小，递归下去，最终子问题中的 ddd 一定会变为 000 （最高位一直在变小），此时有 x=p[k]x=p[k]x=p[k]，再递归回去，则一定可以得到 xxx 的表达式，证毕（其实这个性质一眼就可以看出来）。

\quad\quad 性质二：对于一个数集 SSS，一定存在一组所含所有数的二进制最高位都不同的线性基。

  • \quad\quad 证明：利用反证法，假设不存在一组所含所有数的二进制最高位都不同的线性基。我们设 PPP 为 SSS 的线性基，按照假设，PPP 中至少存在一对 a,ba,ba,b 满足 aaa 的最高位等于 bbb 的最高位，我们令 c=a⊕bc=a\oplus bc=a⊕b，显然，我们从 PPP 中拿出 aaa ，放入 ccc，PPP 任然是一组线性基，而 ccc 的最高位是比 aaa 要小的，我们不妨设为 k′k'k′，如果线性基中不存在另一个最高位为 k′k'k′ 的线性基，那么，进行这么一系列操作后，我们得到了一组所含所有数的二进制最高位都不同的线性基，假设错误；如果线性基中存在另一个最高位为 k′k'k′ 的线性基，我们就按照证明一的思路递归处理，如果到最后仍然找不到一个空位，那么 aaa 根本就是多余的， PPP 不满足线性基定义的第二要求，故假设错误。

\quad 应用与例题：

\quad\quad 例题：P3812 【模板】线性基。

\quad\quad 题目大意：给定n个整数（数字可能重复），求在这些数中选取任意个，使得他们的异或和最大。

\quad\quad 看到题目，我们就知道，这题可以用线性基解决。

\quad\quad 这也是线性基一个最经典的应用：求子序列的最大异或值。

\quad\quad 方法很简单，构造出了线性基后，我们设定答案初始值为0，然后按 kkk 从高到低枚举 P[k]P[k]P[k]（最高位为 kkk 的线性基），假如 P[k]P[k]P[k] 存在，并且答案第 kkk 位为0，就让答案异或上 P[k]P[k]P[k]。这是一个显然的贪心思路，毕竟二进制高位的一要比低位的值更加重要。

\quad\quad 代码：

#include <bits/stdc++.h>

using namespace std;

inline long long read()
{
	long long x = 0, ch = '~';

	while(!isdigit(ch)) ch = getchar();
	while(isdigit(ch)) x = (x << 3) + (x << 1), x += ch - '0', ch = getchar();

	return x;
}

const int N = 60;

int n;
long long a[N], p[N];

int main()
{
	n = read();
	for(register int k = 1; k <= n; k++)
		a[k] = read();

	for(register int k = 1; k <= n; k++)
		for(int i = 55; i >= 0; i--)
			if(a[k] & (1ll << i))
			{
				if(p[i]) a[k] ^= p[i];
				else
				{
					p[i] = a[k];
					break;
				}
			}

	long long maxn = 0;

	for(register int k = 55; k >= 0; k--)
		if(!(maxn & (1ll << k)))
			maxn ^= p[k];

	printf("%lld", maxn);

	return 0;
}
